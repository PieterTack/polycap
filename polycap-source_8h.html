Detailed Description

API for dealing with polycap_source in polycap.

This header contains all functions and definitions that are necessary to create, manipulate and free polycap_source structures as used by polycap. These structures contain information on the source from which photons can be (randomly) selected to illuminate a polycapillary optic.

typedef struct _polycap_source polycap_source

Struct containing information on the source from which photons can be (randomly) selected

When this struct is no longer required, it is the user's responsability to free the memory using polycap_source_free().
